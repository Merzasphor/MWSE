to automaticly create archives

1) goto: ftp://sunsite.icm.edu.pl/pub/unix/archiving/info-zip/WIN32/zip232xN.zip and unpack 'zip.exe' in this directory
2) get Winrar from 'www.rarlabs.com', install, and copy 'rar.exe' from the WinRar directory to this directory
3) run CreateArchives.bat and enjoy!
