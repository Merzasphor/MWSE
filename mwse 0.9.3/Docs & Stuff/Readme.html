The Morrowind Script Extender is a program designed to increase the features available to Morrowind scripters. When run with the game, it adds extra script functions, allowing the creation of plugins that are impossible or at least very difficult while running the game alone.

This version builds upon the 0.9.2 version from cdcooley, it only supports Morrowind version 1.6.1820 (Bloodmoon and GOTY versions).

for any support go to: http://sourceforge.net/projects/mwse, or PM Tp21 or Fliggerty on the forums.

Installing and Running the Program

Simply put the MWSE.exe program into your Morrowind installation directory and run it to start the game. It will start the Morrowind Launcher and then wait for the game to start. It also automatically shuts itself down when Morrowind ends. (You don't have to start the game from the launcher. If you quit the launcher without starting Morrowind, MWSE will keep running until you stop it or start the game some other way.)

Using Other Programs (MWE, FPS Optimizer, MGE) with MWSE

you can bypass the launcher by putting "Morrowind.exe" as an argument to the shortcut (after MWSE.exe). it then starts Morrowind.exe automaticly. you can redirect other programs to start MWSE instead of Morrowind.
