tes3raceHeightWeight
====================================================================================================

A substructure for a race, storing a height or weight for male and female actors.

Properties
----------------------------------------------------------------------------------------------------

`female`_ (`number`_)
    The height or weight for female actors.

`male`_ (`number`_)
    The height or weight for male actors.
