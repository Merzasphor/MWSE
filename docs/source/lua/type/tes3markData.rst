tes3markData
====================================================================================================

A simple structure to hold mark/recall data.

Properties
----------------------------------------------------------------------------------------------------

`cell`_ (`tes3cell`_)
    The cell to recall to.

`position`_ (`tes3vector3`_)
    The position to recall to.

`rotation`_ (`number`_)
    The rotation to use when recalling.
