# tes3raceSkillBonus

A substructure for a race, storing up to 7 skill bonuses.

## Properties

### `bonus`

The value to modify the skill with.

**Returns**:

* `result` (number)

***

### `skill`

The skill id to modify. Maps to values in [`tes3.skill`](https://mwse.github.io/MWSE/references/skills/) namespace.

**Returns**:

* `result` (number)

***
