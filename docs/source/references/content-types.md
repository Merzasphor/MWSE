# Content Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.contentType.*` namespace. For example, `tes3.contentType.layout` has a value of `layout`.

ID       | Content Type
-------- | --------------
layout   | layout
model    | model
rect     | rect
image    | image
text     | text
